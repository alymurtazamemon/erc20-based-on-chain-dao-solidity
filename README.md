# ERC20 based On-Chain DAO - Solidity

## Install Dependencies

```bash
npm install
```
or

```bash
yarn install
```

## Compile Contracts

```bash
npx hardhat compile
```

## Deploy on Hardhat Network

```bash
npx hardhat deploy
```

## Run Hardhat Node

```bash
npx hardhat node
```

## Run Scripts on Hardhat Localhost/Node

```bash
npx hardhat run scripts/1.\ create-a-proposal --network localhost
```
